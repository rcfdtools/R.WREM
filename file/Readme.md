# Course Folder Structure

For the development of the different activities in this course, the following directories are required, which must be contained in the `C:\WREM` or `D:\WREM` folder. [:package:Download.](WREM_FileStructure.zip)

> Note: Do not include files in the repository root; use the corresponding folder. If you need to create temporary files, use the `\temp` folder.

| Folder                                             | Content                                                                                                                                                                                                                                                                                                                                                                                                            |
|:---------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [:open_file_folder:data](data)                     | Folder for storing the base data supplied or downloaded for each workshop or for the project. Do not include bibliographic reference documents, such as open-access books, articles, or compiled technical reports, in this folder. Use the _\ref_ folder instead. This folder should contain subfolders for each source compiled, e.g., ANLA, DANE, ERA5, IDEAM, IGA, NASA, POT, RNEC, SGC, SISBEN, UECIJG, USGS. |
| [:open_file_folder:dem](dem)                       | Folder for storing digital elevation models in GeoTIFF (.tif), TIN, LIDAR (.laz, .las) and other corresponding formats.                                                                                                                                                                                                                                                                                            |
| [:open_file_folder:gdb](gdb)                       | Folder for storing spatial databases or geodatabases. Store files such as .mdb, GDB File, Spatialite, Oracle, GPKG, and other associated files in this container.                                                                                                                                                                                                                                                  |
| [:open_file_folder:graph](graph)                   | Folder for storing graphics, figures, illustrations, and screenshots used generally in this course or in the project submission.                                                                                                                                                                                                                                                                                   |
| [:open_file_folder:grid](grid)                     | Folder for storing raster maps. Store GeoTIFF (.tif), Esri GRID, Ascii (.asc) and other associated files in this container.                                                                                                                                                                                                                                                                                        |
| [:open_file_folder:hec](hec)                       | Folder for storing hydrological and hydraulic models.                                                                                                                                                                                                                                                                                                                                                              |
| [:open_file_folder:map](map)                       | Folder for storing project maps, globe views, and geographic scenes.                                                                                                                                                                                                                                                                                                                                               |
| [:open_file_folder:output](output)                 | Output folder of spatial operations performed using scripts in Python or other languages.                                                                                                                                                                                                                                                                                                                          |
| [:open_file_folder:projectionfile](projectionfile) | Folder for storing coordinate projection files (.prj). Using this folder prevents errors in the development of hydraulic models using HEC-RAS, since these models also use the same extension.                                                                                                                                                                                                                     |
| [:open_file_folder:ref](ref)                       | Folder for storing bibliographic reference documents, including free-use books, articles, and collected technical reports.                                                                                                                                                                                                                                                                                         |
| [:open_file_folder:report](report)                 | Folder for storing reports and technical information on quizzes and exams in Microsoft Word and Adobe Acrobat format.                                                                                                                                                                                                                                                                                              |
| [:open_file_folder:shp](shp)                       | Folder for storing the produced or generated shapefiles (.shp). Remember that each shapefile consists of four files: .shp, .shx, .prj, and .dbf. You can also include the .xml metadata files.                                                                                                                                                                                                                     |
| [:open_file_folder:src](src)                       | Folder for storing scripts in Python and other languages. Also use this directory to store books created with Anaconda's Jupyter.                                                                                                                                                                                                                                                                                  |
| [:open_file_folder:table](table)                   | Folder for storing data tables such as: text files .txt, .asc different from grids, comma-separated text .csv, Microsoft Excel files .xls or .xlsx, DBase File .dbf and non-spatial databases in Microsoft Access .mdb, .accdb or .accde format, sqlite, Oracle.                                                                                                                                                   |
| [:open_file_folder:temp](temp)                     | A folder for the temporary storage of data that can be completely deleted at any time. For example, for conducting preliminary tests or reviewing downloaded files before incorporating them into the official workshop or project folders.                                                                                                                                                                        |
| [:open_file_folder:tool](tool)                     | Folder for storing complementary analysis tools and installers. For example, installation files for HEC tools, external data viewers like Panoply, Python installers, and other free tools.                                                                                                                                                                                                                        |
