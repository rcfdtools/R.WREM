# Modelos hidrológicos e hidráulicos HEC

\hec: carpeta para el almacenamiento de modelos hidrológicos e hidráulicos. 

* HECHMS_v0: Modelo hidrológico en HEC-HMS
* HECRAS_Prototype1D_v0: Modelo hidráulico prototipo 1D de la sección hidráulica del Río Bogotá en la estación Puente La Virgen.
* HECRAS_Prototype2D_v0: Modelo hidráulico prototipo 2D de la sección hidráulica del Río Bogotá en la estación Puente La Virgen.
* HECRAS_River_v0: Modelo hidráulico 1D/2D Río Fucha en HEC-RAS con paso de vía 1D
* HECRAS_River_v0a: Modelo hidráulico 1D/2D Río Fucha en HEC-RAS con paso de vía 2D
